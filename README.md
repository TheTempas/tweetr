# Tweeter Project

Tweeter is a simple, single-page AJAX-based Twitter clone that uses jQuery, HTML5, and CSS3.

## Screenshot

!["Screenshot of Tweetr"](https://github.com/TheTempas/tweetr/blob/master/docs/Tweetr.png)

### Dependencies

- Body-Parser
- Chance
- Express
- MD5
- Moment
- MongoDB
- Nodemon
